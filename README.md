# FreeTV---iOS-E2E-QA-Project-STR-STD-STP-SRS
Full-cycle manual QA portfolio project for the FreeTV iOS streaming app (v1.25.32, iPhone 14 Pro). IEEE 830 SRS, IEEE 829 STP, STD and STR. 123 test cases across Functional, Non-Functional, API and Database suites. 9 JIRA defects. API testing via Fiddler and Postman.
